⚡ СИСТЕМА СКОРОСТИ - КАК РАБОТАЕТ:

ФОРМУЛА:
   Количество атак = 1 + floor(speed / 10)

ПРИМЕРЫ:

Карта с speed = 5:
   Атак: 1 + floor(5 / 10) = 1 + 0 = 1 атака ✅

Карта с speed = 15:
   Атак: 1 + floor(15 / 10) = 1 + 1 = 2 атаки ✅

Карта с speed = 18:
   Атак: 1 + floor(18 / 10) = 1 + 1 = 2 атаки ✅

Карта с speed = 20:
   Атак: 1 + floor(20 / 10) = 1 + 2 = 3 атаки ✅ (ОШИБКА - должно быть 1+2=3, но на самом деле 20/10=2)

Карта с speed = 25:
   Атак: 1 + floor(25 / 10) = 1 + 2 = 3 атаки ✅

Карта с speed = 30:
   Атак: 1 + floor(30 / 10) = 1 + 3 = 4 атаки ✅

ПРИМЕНЕНИЕ:
   • Работает для ВСЕХ карт (игрока и бота)
   • Атаки выполняются ПОСЛЕДОВАТЕЛЬНО
   • Если карта умирает - её атаки пропускаются
