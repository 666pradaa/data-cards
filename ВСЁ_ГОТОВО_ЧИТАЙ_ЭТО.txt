ПРОБЛЕМА: permission_denied - не можете войти в аккаунт

РЕШЕНИЕ: Используйте ПРОСТЫЕ правила Firebase

1. Откройте: https://console.firebase.google.com/
2. Выберите проект: dota-cards
3. Realtime Database → Rules
4. Удалите ВСЁ и вставьте ЭТО:

{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}

5. Нажмите "PUBLISH"
6. Подождите 10-15 секунд
7. Обновите игру (F5)

✅ Теперь вход должен работать!

Админка теперь открывается ПО ПРОМОКОДУ!

КАК ПОЛУЧИТЬ ДОСТУП:
---------------------

1. Войдите в свой аккаунт
2. Откройте профиль
3. Введите промокод: ADMINPANEL666
4. Нажмите "Использовать"
5. Увидите: "Админ доступ получен!"
6. В главном меню появится кнопка "⚙️ Админ"
7. Нажмите на нее - откроется админ-панель

ОСОБЕННОСТИ:
-------------

✅ Промокод привязывается к АККАУНТУ (не к устройству)
✅ Можно входить с любых устройств с админ-правами
✅ Работает один раз, права остаются навсегда
✅ Пароля больше НЕТ - только промокод!
