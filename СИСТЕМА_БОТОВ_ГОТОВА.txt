═══════════════════════════════════════════════════════════════
  🆕 НОВЫЕ ФУНКЦИИ
═══════════════════════════════════════════════════════════════

1. startBotTurn()
   → Начало хода бота
   → Блокирует клики игрока
   → Помечает карты на кулдауне ⏳
   → Через 0.8 сек → selectBotAttacker()

2. selectBotAttacker() - НОВАЯ!
   → Выбирает доступную карту бота
   → Подсвечивает желтым 🟡
   → Показывает "CardName атакует!"
   → Через 0.8 сек → selectBotTarget()

3. selectBotTarget() - НОВАЯ!
   → Выбирает случайную цель
   → Подсвечивает красным 🔴
   → Показывает "CardName атакует TargetName!"
   → Через 0.8 сек → атака

4. performBotMultipleAttacks()
   → Выполняет атаки
   → С учетом скорости карты
   → Логирует каждую атаку
   → Переключает цель если умирает

-------

→ Атака 1

3.6 сек (если скорость >= 20)
-------
→ Атака 2

4.8 сек (если скорость >= 30)
-------
→ Атака 3

   → Зеленые карты доступны 🟢
   → Можете выбрать карту

   → Недоступна для выбора

ЕСЛИ ЧТО-ТО НЕ ТАК:
--------------------

❌ Бот не ходит
   → Проверьте консоль
   → Должно быть "🤖 ХОД БОТА НАЧАЛСЯ"
   → Если нет - напишите мне

❌ Нет подсветки
   → Обновите страницу (F5)
   → Проверьте что CSS загружен
   → Должны быть классы .selected, .target-available

❌ Бот ходит мгновенно
   → Старая версия кода
   → Обновите страницу (F5)
   → Должны быть паузы 0.8 сек

❌ Нет логов в консоли
   → Консоль не открыта?
   → Откройте F12
   → Должны быть эмодзи 🤖🟡🔴⚔️
