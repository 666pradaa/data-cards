ШАГ 1: Выдели ВСЁ между линиями ниже
ШАГ 2: Скопируй (Ctrl+C / Cmd+C)
ШАГ 3: Firebase Console → Realtime Database → Rules
ШАГ 4: Удали всё что там есть
ШАГ 5: Вставь скопированное (Ctrl+V / Cmd+V)
ШАГ 6: Нажми "PUBLISH"

ПРОВЕРКА:
---------

После вставки вы должны видеть РОВНО ЭТО:

{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}

Если видите что-то другое - удалите и вставьте снова

═══════════════════════════════════════════════════════════════

ЧТО ДАЛЬШЕ:
-----------

1. Нажмите "PUBLISH" (большая зеленая кнопка)
2. Подождите 10 секунд
3. Обновите игру (F5)
4. Войдите в аккаунт

✅ Должно работать!
