🍎 SAFARI (родной браузер macOS)
──────────────────────────────────────────────────────────────────────────

СНАЧАЛА ВКЛЮЧИТЕ МЕНЮ РАЗРАБОТЧИКА:
1. Откройте Safari
2. Вверху: Safari → Настройки (Settings)
3. Вкладка "Дополнения" (Advanced)
4. ✅ Включите: "Показывать меню «Разработка» в строке меню"

ТЕПЕРЬ ОТКРОЙТЕ CONSOLE:
⌨️ Способ 1: Cmd + Option + C
⌨️ Способ 2: Меню "Разработка" → "Показать веб-инспектор"
⌨️ Способ 3: Правой кнопкой на странице → "Проверить элемент"

═══════════════════════════════════════════════════════════════════════════

🌐 GOOGLE CHROME
──────────────────────────────────────────────────────────────────────────

⌨️ Способ 1: Cmd + Option + J (сразу Console)
⌨️ Способ 2: Cmd + Option + I (DevTools, потом вкладка Console)
⌨️ Способ 3: Меню ⋮ → More Tools → Developer Tools
⌨️ Способ 4: Правой кнопкой → "Проверить" (Inspect)

═══════════════════════════════════════════════════════════════════════════

🦊 FIREFOX
──────────────────────────────────────────────────────────────────────────

⌨️ Способ 1: Cmd + Option + K (сразу Console)
⌨️ Способ 2: Cmd + Option + I (DevTools)
⌨️ Способ 3: Меню → Web Developer → Web Console
⌨️ Способ 4: Правой кнопкой → "Исследовать элемент"

═══════════════════════════════════════════════════════════════════════════

🎯 САМЫЙ ПРОСТОЙ СПОСОБ (РАБОТАЕТ ВЕЗДЕ):
──────────────────────────────────────────────────────────────────────────

1. Откройте index.html в браузере
2. НАЖМИТЕ ПРАВОЙ КНОПКОЙ МЫШИ на странице
3. Выберите "Inspect" / "Проверить" / "Исследовать элемент"
4. Появится панель снизу или сбоку
5. Найдите вкладку "Console" и нажмите на неё

💡 АЛЬТЕРНАТИВА БЕЗ CONSOLE:
──────────────────────────────────────────────────────────────────────────

Можно тестировать БЕЗ Console!

1. Просто зарегистрируйтесь в игре
2. Откройте Firebase Console в браузере:
   https://console.firebase.google.com/
3. Выберите проект "dota-cards"
4. Authentication → должен появиться пользователь
5. Realtime Database → должны быть данные

Если там есть данные - FIREBASE РАБОТАЕТ! ✅

🎮 ТЕСТ БЕЗ CONSOLE:
