═══════════════════════════════════════════════════════════════
  📱 ПОШАГОВАЯ ИНСТРУКЦИЯ
═══════════════════════════════════════════════════════════════

ШАГ 1: Firebase Console
→ console.firebase.google.com
→ Войди в Google аккаунт

ШАГ 2: Выбери проект
→ Кликни на "dota-cards"

ШАГ 3: Открой Realtime Database
→ Слева меню: Build
→ Кликни: Realtime Database

ШАГ 4: Открой Rules
→ Сверху вкладки: [Data] [Rules] [Backups]
→ Кликни: [Rules]

ШАГ 5: Замени правила
→ Выделить всё: Ctrl+A (Cmd+A на Mac)
→ Удалить: Delete
→ Вставить правила из этого файла (5 строк)

ШАГ 6: Опубликуй
→ Кнопка "PUBLISH" справа сверху
→ Кликни на неё
→ Подожди пока станет неактивной

ШАГ 7: Проверь
→ Должно показать "Rules published"
→ Проверь что в поле ровно 5 строк как выше

ШАГ 8: Подожди
→ 15-20 секунд (Firebase применяет правила)

ШАГ 9: Обнови игру
→ F5 или Ctrl+R (Cmd+R на Mac)
→ Или закрой и открой вкладку

ШАГ 10: Войди
→ Введи логин и пароль
→ Нажми "Войти"
→ Должно работать!

Если после всего вход не работает:

1. Очисти кеш браузера
   → Chrome: Ctrl+Shift+Delete
   → Safari: Cmd+Option+E
   → Firefox: Ctrl+Shift+Delete

2. Перезагрузи браузер
   → Закрой полностью
   → Открой снова

3. Попробуй другой браузер
   → Chrome
   → Firefox
   → Safari
   → Edge

4. Попробуй режим инкогнито
   → Ctrl+Shift+N (Chrome)
   → Cmd+Shift+N (Safari)
