СПОСОБ 1: АВТОМАТИЧЕСКИЙ СКРИПТ (РЕКОМЕНДУЮ)
---------------------------------------------

1. Создайте репозиторий на GitHub:
   → https://github.com/ → New repository
   → Название: dota-cards
   → НЕ ставьте галочки!
   → Create repository
   → Скопируйте ссылку (https://github.com/USERNAME/dota-cards.git)

2. Двойной клик по файлу:
   → upload_to_github.command

3. Введите скопированную ссылку

4. Введите описание: "Initial commit - DOTA CARDS Game"

5. ГОТОВО! ✅
