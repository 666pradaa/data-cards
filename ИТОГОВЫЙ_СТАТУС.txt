⚠️ НЕ ЗАБУДЬТЕ: FIREBASE RULES

Добавьте в Firebase Console → Realtime Database → Rules:

```json
{
  "rules": {
    "users": {
      ".read": "auth != null",
      ".write": false,
      "$uid": {
        ".write": "$uid === auth.uid"
      }
    },
    "clans": {
      ".read": "auth != null",
      ".write": false,
      "$clanId": {
        ".write": "auth != null"
      }
    },
    "rooms": {
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
```

Ссылка: https://console.firebase.google.com/project/dota-cards/database
